# Quiz Manager - Sprachdateien

## Deutsch (Standard)

Das Quiz Manager Modul ist **nativ auf Deutsch** entwickelt. Alle Texte, Meldungen und Beschriftungen sind bereits in deutscher Sprache.

### Enthaltene deutsche Texte:

- **Management-Interface:** "Quiz Manager", "Neue Kategorie hinzufügen", "Bearbeiten", "Löschen"
- **Formular-Labels:** "Titel", "Beschreibung", "Kategorie", "Veröffentlichungsdatum"
- **Frontend-Anzeige:** "Bestandene Quizze", "Bescheinigung ausdrucken"
- **Fehlermeldungen:** "Keine Berechtigung", "Ungültige Anfrage"
- **Erfolgsmeldungen:** "Kategorie erfolgreich aktualisiert", "Daten erfolgreich an Zoho gesendet"

## Übersetzung in andere Sprachen

Falls Sie das Modul in eine andere Sprache übersetzen möchten:

### Option 1: Poedit verwenden

1. **quiz-manager-de_DE.pot** in Poedit öffnen
2. Neue Übersetzung erstellen (z.B. Englisch: en_US)
3. Alle Strings übersetzen
4. Als .po und .mo Datei speichern in diesem Ordner

### Option 2: Loco Translate Plugin

1. WordPress Admin → Loco Translate
2. Plugin "Quiz Manager" auswählen
3. Neue Sprache anlegen
4. Strings übersetzen
5. Speichern

## Hinweis

Das Modul verwendet **keine** WordPress Internationalisierung-Funktionen wie `__()` oder `_e()`, da es ausschließlich für deutschsprachige Websites entwickelt wurde.

Für professionelle mehrsprachige Unterstützung müsste der PHP-Code entsprechend angepasst werden.
